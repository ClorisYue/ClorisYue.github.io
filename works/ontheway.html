On The Way

Description:

On The Way is the Android project of Mobile Application Development (01.2016) finished by 2 members. It is a mobile APP aims to accompany drivers and alert them with potential danger. It mainly has following functions:
Loading local photos to the gallery or taking photos
Marking drive routes on Baidu Map and record the distance and speed

Developing functions:
Vehicle detection on static photos
Real-time vehicle detection with camera

Structure


Environment : Android Studio(APP), Visual Studio&Matlab(Vehicle detection)


APP Runtime screenshot


Vehicle detection

The detection on static pictures is realized by refined SubCat method. Based on the open Matlab source on KITTI, we rewrote the detection part into c++ on Visual Studio and get the following detection effect that can recognize and mark vehicles in the picture.


However the computation needs a high CPU and GPU performance, thus further development is needed to apply the detection into mobile phones.

00. What's ECC-ATS?

ECC-ATS - Electronic Control Cloud Auto Test System

Designed for testers to test the functionality of electronic control board automatically, manage, check and analyze the testing statistics. It contains tester box, EA-control client and EA-control server.

View Version0 Demo

Fig. ECC-ATS Structure

Electronic control board is the core control module of household appliances.
Tester box is provided by ECC, connecting with the electronic control board under testing. It receives executable testing script and generates electronic signal to simulate how to control the board by pressing buttons.And it receives outputs of the control board and returns these statistics to sever and be visualized on the website.

03. Real-time Data Visualization

As mentioned in the information architecture section, there are four tyoes of content in the detailed information page. The most important section is the interactive real-time waveform chart. Statistics can be divided into analog and digital signals. For digital signal, we used square wave referring to the oscilloscope. And considering their needs of:

  • Multiselecting and presenting these signals in order.
  • Linking multiple charts to compare the relationship among some signals.
  • Measuring pulse width, periods, frequency, duty circle and other related statistics.

The last part is test report, every row represents a step from the uploaded test execl, including input parameters and constrains which are used for deciding whether this step is passed. Also, corresponding outputs should be fill into each row. And this table can be exported into an editable Excel file.

04. Exception Analysis

Table of exception analysis

How to deal with execeptions is important to provide stable user experience. After analysis, execptions are most possible occur between the client and server side, and the tester box and server due to communication interruption. I designed execption solutions considering reducing user operations, and allowing user to notice and control the situation.
